# Support

If you experience issues while using the application, please contact support with the following information when possible:

- A description of the issue
- Steps to reproduce the problem
- Screenshots or error messages
- Browser and device information

Response times are not guaranteed. Support is provided on a best-effort basis.

## Security Issues

If you discover a security vulnerability, please report it responsibly and avoid publicly disclosing sensitive details before the issue has been addressed.

## Availability

Support availability may change without notice. We may modify, suspend, or discontinue support services at any time.
